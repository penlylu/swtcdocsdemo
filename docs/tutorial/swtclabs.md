# SWTC Labs

***

SWTC Labs 是一个面向用户展示SWTC公链正在开发的新功能以及收集测试意见的平台，SWTC基金会将针对用户的意见与建议，对这些创新功能和服务进行优化。

## Service节点

功能：与 Public Node 相连，提供所有的账本的实时同步功能；提供一套全新的访问接口，可以对外提供SWTC公链相关的API与RPC调用。
